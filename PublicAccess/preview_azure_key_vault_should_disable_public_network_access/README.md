# [Preview]: Azure Key Vault should disable public network access

## Display Name

[Preview]: Azure Key Vault should disable public network access

## Mode

`All`

## Description

Disable public network access for your key vault so that it's not accessible over the public internet. This can reduce data leakage risks.
